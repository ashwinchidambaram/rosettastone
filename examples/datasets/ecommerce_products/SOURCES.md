# Ecommerce Products Dataset — Sources

## Overview
300 product description generation tasks across 6 product categories. Input: structured product spec. Output: marketing description (80-150 words, 3-5 bullets, "Perfect for..." closing).

## Data Sources

### Synthetic Generation
All product specs are synthetically generated using custom Faker-based templates with domain-specific vocabulary.

**Seed inspiration**: DummyJSON (https://dummyjson.com, MIT license) — product names and categories used as conceptual seeds, all content independently authored.

## Output Format Requirements
- 80–150 words
- 3–5 bullet points
- No superlatives (best, amazing, revolutionary, etc.)
- Ends with "Perfect for..." sentence
