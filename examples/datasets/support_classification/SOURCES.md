# Support Classification Dataset — Sources

## Overview
400 labeled customer support tickets across 9 categories, used for classification model migration testing.

## Data Sources

### Bitext Customer Support LLM Chatbot Training Dataset
- **License**: CDLA-Sharing-1.0
- **URL**: https://huggingface.co/datasets/bitext/Bitext-customer-support-llm-chatbot-training-dataset
- **Usage**: Source tickets for billing, account, shipping, cancellation, feedback, product categories
- **Label mapping**:
  - PAYMENT/INVOICE/REFUND intents → `billing`
  - ACCOUNT intents → `account`
  - SHIPPING_ADDRESS/DELIVERY intents → `shipping`
  - cancel_order/check_cancellation_fee → `cancellation`
  - complaint/review → `feedback`
  - place_order/change_order/track_order → `product`

### Kaggle Customer Support Ticket Dataset (suraj520)
- **License**: CC0 Public Domain
- **URL**: https://www.kaggle.com/datasets/suraj520/customer-support-ticket-dataset
- **Usage**: Source tickets for `technical` category (Ticket Type = "Technical issue")

### Synthetic Generation
- **Usage**: `spam` category (25 tickets) and `ambiguous` cross-category tickets (35 tickets)
- **Method**: Handcrafted examples covering spam patterns and genuine cross-category ambiguity
